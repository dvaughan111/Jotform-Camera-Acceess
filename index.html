JotForm PDF Thumbnail Solution

Complete implementation for displaying clickable image thumbnails in your PDF documents

The Problem: Images Not Showing in PDF

When using JotForm, you might encounter an issue where images appear as [object Object] or blank spaces in your PDF output instead of proper thumbnails.

This happens because JotForm requires a specific data structure to properly render images in PDFs.

The Solution

To ensure images display as clickable thumbnails in your PDF, you need to provide JotForm with properly formatted data that includes:

1. Correct Data Structure

Each image must be submitted with these properties:

  • url: The image URL or base64 data
  • fileName: A descriptive filename
  • pdfOptions: Special instructions for PDF rendering

2. Proper PDF Options

The pdfOptions object must include these properties:

  • display: 'thumbnail' - Shows as thumbnail instead of link
  • width: 'auto' and height: 'auto' - Maintains aspect ratio
  • alignment: 'center' - Centers the thumbnail
  • border: '1px solid #ccc' - Adds a subtle border
  • padding: '5px' - Adds spacing around the thumbnail

3. Stringified JSON

You must stringify your image data array using JSON.stringify() before submitting to JotForm.

Implementation Code

// Sample image data structure for JotForm
const imageData = [
  {
    url: 'data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEASABIAAD...',
    fileName: 'product-photo-12345.jpg',
    pdfOptions: {
      display: 'thumbnail',
      width: 'auto',
      height: 'auto',
      alignment: 'center',
      border: '1px solid #ccc',
      padding: '5px'
    }
  },
  // More images...
];

// How to submit to JotForm
const submitData = {
  valid: true,
  value: JSON.stringify(imageData)
};

// Send to JotForm
JFCustomWidget.sendSubmit(submitData);

Step-by-Step Implementation

1. Capture or Upload Images

Use a camera widget or file uploader that provides image data as base64 strings or URLs.

2. Structure Your Data Correctly

For each image, create an object with the required properties shown in the code example.

3. Add PDF Options

Include the pdfOptions object with thumbnail display settings for each image.

4. Stringify Before Submission

Convert your array of image objects to a JSON string before sending to JotForm.

5. Test Your Implementation

Submit your form and verify that the PDF output shows clickable thumbnails.

Troubleshooting Tips

  • Verify that each image object has all required properties (url, fileName, pdfOptions)
  • Ensure you're using JSON.stringify() on your image array
  • Check that your base64 image data is properly formatted
  • Test with a simple example first before implementing complex logic
  • Consult JotForm's documentation for any updates to their PDF rendering
