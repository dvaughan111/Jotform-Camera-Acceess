Fullscreen Violation

Violation Message

[Violation] Potential permissions policy violation: fullscreen is not allowed in this document.

This violation occurs when your code attempts to use the Fullscreen API without having the proper permissions policy enabled.

Solution

Add the following permissions policy header to your server response:

Permissions-Policy: fullscreen=(self "https://yourdomain.com")

Or in your HTML meta tag:

<meta http-equiv="Permissions-Policy" content="fullscreen=(self)">
!

Autoplay Violation

Violation Message

[Violation] Potential permissions policy violation: autoplay is not allowed in this document.

This violation occurs when media tries to autoplay without the proper permissions policy or user interaction.

Solution

Add the following permissions policy header to your server response:

Permissions-Policy: autoplay=(self "https://yourdomain.com")

Or ensure autoplay is only triggered after user interaction:

document.getElementById('playButton').addEventListener('click', () => {
  document.getElementById('myVideo').play();
});

Encrypted-Media Violation

Violation Message

[Violation] Potential permissions policy violation: encrypted-media is not allowed in this document.

This violation occurs when your page tries to use the Encrypted Media Extensions (EME) API without proper permissions.

Solution

Add the following permissions policy header to your server response:

Permissions-Policy: encrypted-media=(self "https://yourdomain.com")

Or in your HTML meta tag:

<meta http-equiv="Permissions-Policy" content="encrypted-media=(self)">
